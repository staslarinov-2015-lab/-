# Family Expense Bot

MVP Telegram-бот для семейного учета расходов.

Что умеет:

- принимать траты обычным текстом;
- определять сумму и категорию;
- сохранять расходы в SQLite;
- показывать сводки за день, неделю и месяц;
- автоматически отправлять ежедневный, еженедельный и ежемесячный отчеты в один чат.

## Пример сообщений

- `1200 продукты`
- `450 такси`
- `Потратили 3150 на аптеку`
- `799 подписка на Яндекс`

## Команды

- `/start` - приветствие
- `/help` - подсказка по формату
- `/today` - сводка за сегодня
- `/week` - сводка за последние 7 дней
- `/month` - сводка за текущий месяц
- `/categories` - список категорий
- `/chatid` - показать id текущего чата

## Быстрый старт

1. Создайте бота через BotFather и получите токен.
2. Скопируйте `.env.example` в `.env`.
3. Укажите `BOT_TOKEN`.
4. Для первого запуска можно временно оставить `ALLOWED_CHAT_ID` пустым.
5. Установите зависимости:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

6. Запустите бота:

```bash
python3 -m bot.main
```

## Как получить chat id

После запуска добавьте бота в нужный семейный чат и отправьте команду `/chatid`. Бот вернет текущее значение `chat_id`, которое нужно вставить в `ALLOWED_CHAT_ID` в файле `.env`. После этого перезапустите бота.

## Архитектура

- `bot/main.py` - Telegram-бот и обработчики
- `bot/config.py` - настройки из переменных окружения
- `bot/db.py` - SQLite и запросы
- `bot/categorizer.py` - правила категоризации
- `bot/reports.py` - логика отчетов и аналитики
- `bot/scheduler.py` - планировщик авто-отчетов
